# 🛍️ Nuvara – E‑commerce Website

Nuvara is a fully responsive e‑commerce frontend application built with **React**, **Redux Toolkit**, and **React Bootstrap**. It integrates with a RESTful API to fetch products, manage a shopping cart, and process orders.

---

## ✨ Features

- 🏠 **Product listing**
- 🛒 **Shopping cart**
- 👤 **User authentication**
- 💳 **Checkout flow**
- 📱 **Fully responsive**
- 🎨 **React Bootstrap**
- 🧠 **Redux Toolkit**
- 🌐 **REST API integration**
- 🔒 **Protected routes**

---

## 🧰 Tech Stack

| Layer       | Technology                     |
| ----------- | ------------------------------ |
| Frontend    | React 18 + React Router DOM    |
| State       | Redux Toolkit (slices, thunks) |
| UI Library  | React Bootstrap + Scss         |
| HTTP Client | Axios / Fetch API              |
| Hosting     | Vercel                         |
